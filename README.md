Read-only WooCommerce admin report for orders paid through the Snappay gateway.

== Description ==

Ashplug Orders Report for Snappay adds a focused, read-only admin report for WooCommerce stores that use the Snappay payment gateway.

This plugin is independently developed and is not affiliated with, endorsed by, or sponsored by Snappay or WooCommerce.

Features:

* Shows WooCommerce orders paid through the Snappay gateway with pagination
* Displays the total number of Snappay orders in the store
* Displays the date of the first Snappay order in the store
* Highlights cancelled, fully refunded, and partially refunded orders with lightweight visual indicators
* Includes English as the default language and Persian (fa_IR) translation files
* Read-only: does not modify orders or send data externally

The plugin also includes a non-intrusive Pro Features page inside its own admin menu. It only links to information about a separate Pro plugin and does not include any Pro plugin code or dashboard-wide notices.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/ashplug-orders-report-for-snappay/` or install it from the Plugins screen.
2. Activate the plugin.
3. Make sure WooCommerce and a compatible Snappay payment gateway plugin are installed and active.
4. In wp-admin, open the **Ashplug Snappay** menu.

== Frequently Asked Questions ==

= Does this plugin modify order data? =
No. This plugin is strictly read-only.

= Is WooCommerce required? =
Yes. WooCommerce must be installed and active.

= Is the Snappay gateway plugin required? =
Yes. The plugin works only when a compatible Snappay gateway is available.

= Is this an official Snappay or WooCommerce plugin? =
No. This plugin is independently developed and is not affiliated with, endorsed by, or sponsored by Snappay or WooCommerce.

= Is there a Pro plugin? =
Yes. A separate Pro plugin is available outside WordPress.org for advanced financial workflows such as installment schedules, commission calculations, Jalali date filters, and export tools. No Pro plugin code is included in this plugin hosted on WordPress.org.
